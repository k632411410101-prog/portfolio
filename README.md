# Nguyễn Thị Khánh Ly - Portfolio Game

An interactive pixel-art farming game portfolio inspired by Stardew Valley and Harvest Moon.

## 🎮 Features

- **Pixel-Art Game World**: Explore a living, breathing farm environment
- **Smooth Horizontal Scrolling**: Seamless navigation through different sections
- **Dynamic Day/Night Cycle**: Lighting transitions throughout the journey
- **Interactive Elements**: Hover and click animations on buildings and objects
- **Vietnamese Content**: Full Vietnamese language support with proper typography
- **Responsive Design**: Works on all device sizes
- **Ambient Animations**: Floating particles, swaying trees, moving clouds
- **Performance Optimized**: 60 FPS target, lazy-loaded assets

## 🛠️ Tech Stack

- **Framework**: Next.js 15 (App Router)
- **UI Library**: React 19
- **Styling**: Tailwind CSS + Framer Motion
- **Animations**: GSAP + ScrollTrigger, Framer Motion
- **State Management**: Zustand
- **Typography**: Press Start 2P + Be Vietnam Pro

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
